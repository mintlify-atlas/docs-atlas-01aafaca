DentalConnect API is a Node.js/Express REST API that powers dental clinic operations. It provides endpoints for patient appointment lookups, secure password recovery, and automated email appointment reminders.

## What DentalConnect API does

<CardGroup cols={2}>
  <Card title="Appointment lookups" icon="calendar-check">
    Retrieve a patient's next upcoming appointment, including doctor name, service, and scheduled time.
  </Card>
  <Card title="Password recovery" icon="key">
    Secure token-based password reset flow with 15-minute expiry and bcrypt hashing.
  </Card>
  <Card title="Email reminders" icon="envelope">
    Automated cron job sends appointment reminder emails to patients the day before their visit.
  </Card>
  <Card title="Multi-role user model" icon="users">
    Supports four roles: administrator, doctor, receptionist, and patient.
  </Card>
</CardGroup>

## Base URL

```
http://localhost:4000
```

All API routes are prefixed with `/api`. For example:

```
GET /api/citas/proxima/:id_paciente
POST /api/auth/forgot-password
```

## Health check

Verify the API is running:

```bash
curl http://localhost:4000/api/status
```

```json
{ "status": "API Online", "version": "1.0.0" }
```
